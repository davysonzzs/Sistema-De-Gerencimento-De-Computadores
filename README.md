# Sistema de Gerenciamento de Computadores

Sistema desenvolvido com **propósito educativo** para estudar JavaFX, interfaces gráficas e programação em Java.

---

## 🎯 Sobre o Projeto

O Sistema de Gerenciamento de Computadores permite visualizar e controlar o status de uso dos computadores de um laboratório, sala de aula ou lan house de forma simples e intuitiva.

- Interface gráfica desenvolvida em **JavaFX**
- Marcação rápida de status dos computadores
- Feedback visual claro: **verde** = em uso | **vermelho** = disponível

---

## 🚀 Como Utilizar

1. Baixe a versão mais recente na seção **[Releases](https://github.com/davysonzzs/Sistema-De-Gerencimento-De-Computadores/releases)**.
2. Extraia a pasta.
3. Execute o arquivo **`start.bat`** (Windows).
4. O programa iniciará com **3 computadores cadastrados** para teste.

### Como usar:
- Clique na caixinha do computador para **marcar** ou **desmarcar** como em uso.
- **Verde** → Computador em uso
- **Vermelho** → Computador disponível
